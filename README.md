# Epstein_files_RAG
⚖️ Epstein Files RAG MVP  A public, free-to-access AI research assistant for exploring the **Epstein Files** (`teyler/epstein-files-20k` dataset).  This tool allows you to:  - **Ask questions** about the files and get fact-based answers. - **View Sources**: Every answer is grounded in specific documents, with filenames provided.
